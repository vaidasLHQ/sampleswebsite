# TRNDFY - Resume Context for New Chat

## Project Overview
TRNDFY is a premium royalty-free sample pack e-commerce website built with SolidStart + TypeScript.

## Live Site
https://website-solidstart.pages.dev

## Tech Stack
- **Framework**: SolidStart (SolidJS meta-framework)
- **Language**: TypeScript
- **Styling**: CSS (app.css)
- **Database**: Supabase (auth + storage + orders)
- **Payments**: Stripe
- **Email**: Resend
- **Audio**: Howler.js
- **Deployment**: Cloudflare Pages

## Key Files
- `src/app.tsx` - Root layout with header/footer + TRNDFY logo
- `src/routes/index.tsx` - Homepage (V12 design)
- `src/components/SEO.tsx` - SEO component with Schema.org
- `src/components/SampleRowV12.tsx` - Sample row component
- `src/app.css` - All styles (~10K lines)
- `src/data/samples.ts` - Sample data
- `src/lib/cart.ts` - Cart state management
